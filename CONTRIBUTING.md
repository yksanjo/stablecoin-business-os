# Contributing to Stablecoin Business OS

We love contributions! Here's how to help:

## 🚀 Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/stablecoin-business-os.git`
3. Install dependencies: `npm install`
4. Start the dev server: `npm run dev`

## 🧪 Testing

```bash
# Start server
npm run dev

# Run API tests
curl http://localhost:3001/api/health
```

## 📋 PR Guidelines

- Keep PRs focused on one feature/fix
- Add tests for new functionality
- Update README if changing API behavior
- Use conventional commits (`feat:`, `fix:`, `docs:`, etc.)

## 💡 Ideas for Contributions

- **Web dashboard** — React frontend
- **Wallet adapter** — Phantom/Backpack integration
- **More AI features** — LLM-powered insights
- **Multi-chain support** — Base, Arbitrum
- **Fiat on/off ramps** — Jupiter, Circle API
- **Tax exports** — CSV, QuickBooks integration
- **Mobile app** — React Native
